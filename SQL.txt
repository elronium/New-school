SELECT
*Буквы в кавычках, цифры - без

Вывод всех значений: SELECT * FROM таблица;
Вывод без дубликатов: SELECT DISTINCT столбец FROM таблица;
Вывод нужных строк: SELECT * FROM таблица LIMIT число;
Псевдонимы: SELECT столбец AS новое название столбца FROM таблица;
Дополнительные условия для вывода: SELECT столбец FROM таблица WHERE столбец = "значение из столбца";
Условия для вывода с выбором (И/ИЛИ): 
SELECT столбец FROM таблица WHERE столбец = "значение из столбца" AND столбец = "значение из столбца";
SELECT столбец FROM таблица WHERE столбец = "значение из столбца" OR стобец = "значение из столбца";
Поиск похожих значений: SELECT столбец FROM таблица WHERE столбец LIKE "%..."; 
*положение % - какое-то количество символов, положение _ - один символ
Входят в диапазон: SELECT столбец FROM таблица WHERE столбец BETWEEN 1 AND 9;
Не входят в диапазон: SELECT столбец FROM таблица WHERE NOT столбец BETWEEN 1 AND 9;
Конкретное значение: SELECT столбец FROM таблица WHERE столбец IN (1,6,8);
Объединение функций: SELECT столбец FROM таблица WHERE (столбец >= значение OR столбец >= значение) AND столбец = значение;
Сортировка: SELECT столбец FROM таблица ORDER BY столбец для сортировки;
В обратном порядке: SELECT столбец FROM таблица ORDER BY столбец для сортировки DESC;
*стандартная сортировка по-умолчанию ASC

Агрегатные функции:
Среднее значение: SELECT AVG(столбец) FROM таблица;
Сумма: SELECT SUM(столбец) FROM таблица;
Минимальное/максимальное: SELECT MIN(столбец) FROM таблица; / SELECT MAX(столбец) FROM таблица;
Количество записей заполненных значениями: SELECT COUNT(столбец) FROM таблица;
